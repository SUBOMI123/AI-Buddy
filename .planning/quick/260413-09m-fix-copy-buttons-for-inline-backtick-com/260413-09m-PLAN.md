   summary row (collapsed by default); clicking expands/collapses that exchange.

Purpose: Restore copy-button usability after the backtick formatting change; reduce


Actually, since submitIntent slices sessionHistory to max 3 items and the indices

is called (i.e. inside submitIntent where the history push happens, call
`setExpandedHistoryItems(new Set())` right after `setSessionHistory(...)`).

**Rendering — replace the SessionFeed `sessionHistory` prop render:**

The `<SessionFeed>` component currently receives `sessionHistory={sessionHistory()}`
and renders all prior exchanges as flat muted text. Replace that with an inline
`<For>` loop directly in SidebarShell that renders collapsible rows. Pass
`sessionHistory={[]}` to SessionFeed (empty — current streaming exchange only) and
render prior history separately above it.

Build collapsible history rows immediately before the `<SessionFeed>` block inside
the existing `<Show when={... sessionHistory().length > 0 ...}>` guard:

```tsx
<For each={sessionHistory()}>
  {(exchange, index) => {
    const isExpanded = () => expandedHistoryItems().has(index());
    const toggleExpanded = () => {
      setExpandedHistoryItems((prev) => {
        const next = new Set(prev);
        if (next.has(index())) next.delete(index());
        else next.add(index());
        return next;
      });
    };

    // Collapsed summary: truncated intent + chevron
    const summary = exchange.intent.length > 40
      ? exchange.intent.slice(0, 40) + "\u2026"
      : exchange.intent;

    return (
      <div style={{ "border-bottom": "1px solid var(--color-border)" }}>
        {/* Collapsed header row — always visible */}
        <button
          onClick={toggleExpanded}
          aria-expanded={isExpanded()}
          style={{
            width: "100%",
            display: "flex",
            "align-items": "center",
            "justify-content": "space-between",
            background: "transparent",
            border: "none",
            padding: "var(--space-sm) 0",
            cursor: "pointer",
            color: "var(--color-text-secondary)",
            "font-size": "var(--font-size-label)",
            "text-align": "left",
            gap: "var(--space-xs)",
          }}
        >
          <span style={{
            overflow: "hidden",
            "text-overflow": "ellipsis",
            "white-space": "nowrap",
            flex: "1",
          }}>
            {summary}
          </span>
          <span style={{ "flex-shrink": "0", "font-size": "10px" }}>
            {isExpanded() ? "\u25BC" : "\u25B6"}
          </span>
        </button>

        {/* Expanded body — only rendered when expanded */}
        <Show when={isExpanded()}>
          <div style={{
            "font-size": "var(--font-size-label)",
            "line-height": "var(--line-height-body)",
            color: "var(--color-text-secondary)",
            padding: "0 0 var(--space-sm) 0",
            "white-space": "pre-wrap",
            "word-break": "break-word",
          }}>
            {exchange.guidance}
          </div>
        </Show>
      </div>
    );
  }}
</For>
```

This renders above the `<SessionFeed>` which now only receives the active streaming
exchange. Pass `sessionHistory={[]}` to SessionFeed so it no longer renders history
— it only handles the live streaming text display.
