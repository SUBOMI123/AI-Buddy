Human verification checkpoint. No code changes -- verify the complete end-to-end AI guidance loop works as specified.

**Prerequisites:**
1. Ensure Worker is running locally: `cd worker && npx wrangler dev`
2. Ensure Worker has ANTHROPIC_API_KEY set (either in .dev.vars or as Wrangler secret)
3. Ensure APP_HMAC_SECRET matches between worker/.dev.vars and src-tauri/.cargo/config.toml
4. Start the app: `cargo tauri dev`

**Test 1: Basic guidance flow (CORE-01, CORE-02, CORE-03, CORE-04)**
1. Open any unfamiliar application (e.g., Terminal, a text editor, Finder)
2. Toggle the AI Buddy overlay with the global shortcut
3. Type: "How do I create a new folder here?"
4. Press Enter
5. Verify: Three pulsing dots appear (loading state)
6. Verify: Dots disappear and text starts streaming in word-by-word
7. Verify: Guidance includes numbered steps referencing what's visible on screen
8. Verify: Response arrives within ~3-5 seconds of submission

**Test 2: Streaming behavior (D-05, D-06)**
1. Submit another question: "How do I change the font size?"
2. Verify: Previous guidance clears immediately (D-07)
3. Verify: Loading dots reappear
4. Verify: Text appears progressively, not as a single block

**Test 3: Clarification for vague intent (CORE-05)**
1. Submit: "help me"
2. Verify: Claude asks a clarifying question (e.g., "What are you trying to do?")
3. Verify: Claude does NOT generate a guess response

**Test 4: Error handling (D-11)**
1. Stop the Worker (Ctrl+C in wrangler dev terminal)
2. Submit any question
3. Verify: Error message appears inline: "Couldn't reach AI -- check your connection."
4. Verify: A "Retry" button appears below the error
5. Restart the Worker
6. Click Retry
7. Verify: Loading dots appear and guidance streams normally

**Test 5: New question during streaming (D-07)**
1. Submit a long question: "Give me a detailed tutorial on all the keyboard shortcuts in this app"
2. While text is still streaming, submit a new question: "How do I save?"
3. Verify: Previous streaming stops, loading dots appear, new response streams
