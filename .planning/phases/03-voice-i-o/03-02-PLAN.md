# Plan 03-02: Worker Routes + Rust TTS Command

## Goal

Implement the Worker `/stt` and `/tts` routes (replacing Phase 1 stubs), the Rust `cmd_play_tts` command that calls the Worker and plays MP3 audio via rodio, and all TypeScript IPC wrappers for voice preference commands and TTS playback. This completes the server-side and Rust audio pipeline needed for Plan 03-03 frontend wiring.

## Threat Model

### Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Frontend → Worker /stt | Token issuance — must validate HMAC auth or anyone can get AssemblyAI tokens |
| Frontend → Worker /tts | Text forwarding — must validate text length to prevent upstream API abuse |
| Worker → ElevenLabs | TTS request — API key in Wrangler secret, never exposed to client |
| Worker → AssemblyAI | Token fetch — API key in Wrangler secret, never exposed to client |
| Rust play_tts → Worker | Must include x-app-token header so Worker auth middleware passes |

### STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-04 | Elevation of Privilege | Worker /stt route | mitigate | Route is covered by the existing `app.use('*', ...)` HMAC auth middleware already present in worker/src/index.ts. No per-route auth needed — middleware blocks unauthenticated requests before reaching the route handler. Verify middleware applies to POST /stt. |
| T-03-05 | Tampering / DoS | Worker /tts route | mitigate | Validate request body: `text` must be non-empty string, max 2000 characters. Return 400 with `{ error: "text must be 1–2000 characters" }` before calling ElevenLabs. This prevents cost-explosion from large payloads forwarded upstream. |
| T-03-06 | DoS | play_tts concurrent calls | mitigate | D-16: stop current rodio Sink before starting new. Prevents audio overlap and unbounded resource accumulation. Implemented via `Arc<Mutex<Option<Sink>>>` global playback guard. |
| T-03-07 | Information Disclosure | ElevenLabs voice_id | accept | voice_id is a public ElevenLabs model identifier, not a secret. Hardcoding in Worker is acceptable. |

```typescript
app.post('/stt', async (c) => {
  const tokenUrl = 'https://streaming.assemblyai.com/v3/token?expires_in_seconds=300';
  
  let tokenResponse: Response;
  try {
    tokenResponse = await fetch(tokenUrl, {
      method: 'GET',
      headers: {
        'Authorization': c.env.ASSEMBLYAI_API_KEY,
      },
    });
  } catch (err) {
    console.error('AssemblyAI token fetch failed:', err);
    return c.json({ error: 'STT service unavailable' }, 503);
  }

  if (!tokenResponse.ok) {
    console.error('AssemblyAI token endpoint returned', tokenResponse.status);
    return c.json({ error: 'STT token issuance failed' }, 502);
  }

  const body = await tokenResponse.json<{ token: string }>();
  
  if (!body.token) {
    return c.json({ error: 'STT token response missing token field' }, 502);
  }

  return c.json({ token: body.token });
});
```

```typescript
app.post('/tts', async (c) => {
  let body: { text?: string; voice_id?: string };
  try {
    body = await c.req.json();
  } catch {
    return c.json({ error: 'Invalid JSON body' }, 400);
  }

  // T-03-05: Validate text — non-empty, max 2000 chars
  const text = typeof body.text === 'string' ? body.text.trim() : '';
  if (!text) {
    return c.json({ error: 'text must be a non-empty string' }, 400);
  }
  if (text.length > 2000) {
    return c.json({ error: 'text must be 2000 characters or fewer' }, 400);
  }

  // ElevenLabs voice ID — use "Rachel" (natural, clear instructional voice)
  // voice_id: 21m00Tcm4TlvDq8ikWAM is ElevenLabs "Rachel" — clear and neutral
  const voiceId = body.voice_id ?? '21m00Tcm4TlvDq8ikWAM';
  const elevenUrl = `https://api.elevenlabs.io/v1/text-to-speech/${voiceId}/stream`;

  let elevenResponse: Response;
  try {
    elevenResponse = await fetch(elevenUrl, {
      method: 'POST',
      headers: {
        'xi-api-key': c.env.ELEVENLABS_API_KEY,
        'Content-Type': 'application/json',
        'Accept': 'audio/mpeg',
      },
      body: JSON.stringify({
        text,
        model_id: 'eleven_turbo_v2_5',
        voice_settings: {
          stability: 0.5,
          similarity_boost: 0.75,
        },
      }),
    });
  } catch (err) {
    console.error('ElevenLabs request failed:', err);
    return c.json({ error: 'TTS service unavailable' }, 503);
  }

  if (!elevenResponse.ok) {
    const errText = await elevenResponse.text();
    console.error('ElevenLabs returned', elevenResponse.status, errText);
    return c.json({ error: 'TTS generation failed' }, 502);
  }

  // Stream MP3 audio back to caller
  return new Response(elevenResponse.body, {
    status: 200,
    headers: {
      'Content-Type': 'audio/mpeg',
      'Cache-Control': 'no-cache',
      'Transfer-Encoding': 'chunked',
    },
  });
});
```

**Step 1 — Create src-tauri/src/voice/tts.rs**

```rust
//! Text-to-speech playback command.
//!
//! play_tts flow:
//!   1. Validate text (non-empty)
//!   2. POST Worker /tts with { text } and x-app-token header
//!   3. Receive streaming MP3 response
//!   4. Buffer response (Cloudflare Workers stream MP3 in chunks)
//!   5. Decode with rodio Decoder
//!   6. Play via default audio output
//!   7. D-16: stop any existing playback before starting new

use rodio::{Decoder, OutputStream, Sink};
use std::io::Cursor;
use std::sync::{Arc, Mutex};
use tauri::AppHandle;

/// Global playback guard — holds the active Sink so it can be stopped on new playback request.
/// D-16: clicking Play on a second item stops the first.
static ACTIVE_SINK: Mutex<Option<Arc<Sink>>> = Mutex::new(None);

/// Tauri command: fetch TTS audio from Worker and play via rodio.
///
/// Blocks until audio finishes or is stopped. Called from frontend on "Play" button click.
/// Returns Ok(()) when audio starts playing (not when it finishes — fire-and-forget from frontend).
#[tauri::command]
pub async fn cmd_play_tts(app: AppHandle, text: String) -> Result<(), String> {
    // Validate text locally before making network request
    let text = text.trim().to_string();
    if text.is_empty() {
        return Err("text must not be empty".to_string());
    }
    if text.len() > 2000 {
        return Err("text must be 2000 characters or fewer".to_string());
    }

    // D-16: Stop current playback before starting new
    {
        let mut guard = ACTIVE_SINK.lock().map_err(|_| "playback lock poisoned".to_string())?;
        if let Some(sink) = guard.take() {
            sink.stop();
        }
    }

    // Build Worker URL from build-time env var (same as PTT)
    let worker_url = option_env!("WORKER_URL")
        .unwrap_or("http://localhost:8787")
        .to_string();
    let app_token = crate::preferences::cmd_get_token(app.clone());

    // Fetch MP3 audio from Worker /tts
    let client = reqwest::Client::new();
    let response = client
        .post(format!("{}/tts", worker_url.trim_end_matches('/')))
        .header("x-app-token", &app_token)
        .header("Content-Type", "application/json")
        .json(&serde_json::json!({ "text": text }))
        .send()
        .await
        .map_err(|e| format!("TTS request failed: {}", e))?;

    if !response.status().is_success() {
        let status = response.status();
        let err_body = response.text().await.unwrap_or_default();
        return Err(format!("TTS endpoint returned {}: {}", status, err_body));
    }

    // Buffer the full MP3 response
    // Note: Cloudflare Workers stream the response but rodio Decoder needs a seekable reader.
    // Buffer the bytes in memory — ElevenLabs Turbo v2.5 responses for typical guidance
    // text (50–300 chars) are 30–200KB, well within acceptable memory use.
    let audio_bytes = response
        .bytes()
        .await
        .map_err(|e| format!("TTS audio download failed: {}", e))?;

    // Play audio in a std::thread (rodio requires !Send OutputStream on macOS)
    std::thread::spawn(move || {
        let (_stream, stream_handle) = match OutputStream::try_default() {
            Ok(s) => s,
            Err(e) => {
                eprintln!("TTS: audio output unavailable: {}", e);
                return;
            }
        };

        let sink = match Sink::try_new(&stream_handle) {
            Ok(s) => Arc::new(s),
            Err(e) => {
                eprintln!("TTS: failed to create sink: {}", e);
                return;
            }
        };

        // Register sink as active (D-16: allows next Play to stop this one)
        if let Ok(mut guard) = ACTIVE_SINK.lock() {
            *guard = Some(Arc::clone(&sink));
        }

        let cursor = Cursor::new(audio_bytes);
        match Decoder::new(cursor) {
            Ok(source) => {
                sink.append(source);
                sink.sleep_until_end();
            }
            Err(e) => {
                eprintln!("TTS: failed to decode audio: {}", e);
            }
        }

        // Clear the active sink reference when done
        if let Ok(mut guard) = ACTIVE_SINK.lock() {
            *guard = None;
        }
    });

    Ok(())
}
```
