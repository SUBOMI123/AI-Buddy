# AI Buddy

AI Buddy watches your screen and gives you step-by-step guidance to complete any task in
any software — without Googling or getting stuck.

<!-- screenshot: app-screenshot.png -->

## Download

Head to the **[latest release](https://github.com/SUBOMI123/AI-Buddy/releases/latest)** and pick the right file for your platform:

| Platform | Architecture | File to download |
|----------|-------------|-----------------|
| macOS | Apple Silicon (M1/M2/M3/M4) | `AI.Buddy_*_aarch64.dmg` |
| macOS | Intel | `AI.Buddy_*_x64.dmg` |
| Windows | x64 | `AI.Buddy_*_x64-setup.exe` |

If unsure which Mac you have, check **Apple menu → About This Mac → Chip**.

## About

AI Buddy is a Tauri v2 desktop app (macOS + Windows) that watches your screen via screenshot, sends it to Claude AI, and returns step-by-step guidance spoken aloud via ElevenLabs TTS. It is designed to help you complete tasks in unfamiliar software without leaving the app you are already in. This is a closed beta.

Built with [Tauri](https://tauri.app), [SolidJS](https://solidjs.com), and [Claude AI](https://anthropic.com).
